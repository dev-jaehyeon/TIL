Delegating Constructor

- 생성자가 다른 생성자를 호출한다.
- 일반적인 초기화를 대체한다.
- 코드 중복을 막아준다.

```c++
Car::Car() : Car(0)
{
	//totalCount++;
	std::cout << "Car()" << std::endl;
}
Car::Car(float _fuel) : Car(_fuel, 0)
{
	//totalCount++;
	//fuel = _fuel;
	std::cout << "Car(float _fuel)" << std::endl;
}

Car::Car(float amount, int pass)
{
	std::cout << "Car(float amount, int pass)" << std::endl;
	totalCount++;
	fuel = amount;
	speed = 0;
	passengers = 0;
}
```

```c++
#include <iostream>
#include "Car.h"


int main()
{	
	Car car;
	car.Dashboard();
	return 0;
}
```
